##改行  
改行は行末に2つのスペースをいれる。  
```
改行[space][space]
Test
```  
↓
改行  
Test


##強調  
強調はアスタリスクまたはアンダーバーで囲むらしい  
`*ガンガン行こうぜ*` => *ガンガン行こうぜ*  
`**いのちを大事に**` => **いのちを大事に**
